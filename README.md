# Modern React Dashboard (Dark Mode + Responsive)

واجهة لوحة تحكم (Dashboard) عصرية واحترافية مبنية باستخدام **React** و **Tailwind CSS**. تتميز بدعم كامل للوضع الليلي وتصميم متجاوب بالكامل مع جميع الشاشات.

## المميزات
*   **Dark Mode Switch:** تبديل سلس بين الوضع الفاتح والليلي.
*   **Fully Responsive:** تصميم متجاوب.
*   **Professional UI:** قائمة جانبية انسيابية (Slide-in Sidebar) مع تأثيرات زجاجية (Backdrop blur) للهيدر.

## التقنيات المستخدمة
*   **React.js** - المكتبة الأساسية لبناء الواجهات.
*   **Tailwind CSS** - لتنسيق العناصر بسرعة وبنظام Utility-first.
*   **Lucide React** - للأيقونات التقنية العصرية.

##  تشغيل المشروع

1. تحميل المشروع:

```bash
git clone https://github.com/AbdullahAbuZaid04/Dashboard-With-Sidebar.git
```

2. تثبيت المكتبات:

```bash
npm install
```

3. تشغيل المشروع:

```bash
npm start
```
